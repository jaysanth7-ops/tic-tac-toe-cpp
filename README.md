# 🎮 Tic-Tac-Toe Game (C++)

A simple console-based Tic-Tac-Toe game built using C++.

---

## 🚀 Features
- Two-player mode (X vs O)
- Dynamic board display
- Win detection 🏆
- Draw detection 🤝
- Replay option 🔄

---

## 🧠 Concepts Used
- Loops
- Functions
- Arrays (vector)
- Conditional logic

---

## ▶️ How to Run

```bash
g++ main.cpp -o game
./game
